# vue-management-system

> A Vue.js project

## Build Setup

``` bash
# 项目环境
node >= 6.0.0 && npm >= 3.0.0

# 安装依赖
npm install

# 启动服务
npm run dev

# build for production with minification
npm run build
```
## 知识点
* vue vue-router
* webpack
* echarts
* element-ui

## 项目介绍
vue + vue-router 的单页面后台管理系统
